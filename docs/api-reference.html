POST /events

Creates a new event in the system. If eventId is not provided, a UUID will be automatically generated.

Request Headers

Header Value Required
Content-Type application/json Yes

Request Body

Field Type Required Description
title string Yes Event title (non-empty)
description string Yes Event description
date string Yes Event date (YYYY-MM-DD format)
location string Yes Event location (non-empty)
capacity integer Yes Maximum participants (>= 0)
organizer string Yes Organizer name (non-empty)
status string Yes Event status (e.g., "scheduled", "active")
eventId string No Custom event ID (auto-generated if not provided)

Example Request

curl -X POST https://m4vehfzim8.execute-api.us-west-2.amazonaws.com/prod/events \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Tech Conference 2024",
    "description": "Annual technology conference",
    "date": "2024-12-15",
    "location": "San Francisco",
    "capacity": 500,
    "organizer": "Tech Corp",
    "status": "scheduled"
  }'

Response

201 Created

{
  "eventId": "614b345c-5114-4788-bd81-f792d59463b5",
  "title": "Tech Conference 2024",
  "description": "Annual technology conference",
  "date": "2024-12-15",
  "location": "San Francisco",
  "capacity": 500,
  "organizer": "Tech Corp",
  "status": "scheduled"
}

Error Responses

422 Unprocessable Entity - Validation error

{
  "detail": "Validation error message"
}

PUT /events/{eventId}

Updates an existing event. Supports partial updates - only provided fields will be modified, all other fields remain unchanged.

Path Parameters

Parameter Type Description
eventId string Unique event identifier (UUID)

Request Body

All fields are optional. Only include fields you want to update.

Field Type Description
title string Event title (non-empty)
description string Event description
date string Event date (YYYY-MM-DD format)
location string Event location (non-empty)
capacity integer Maximum participants (>= 0)
organizer string Organizer name (non-empty)
status string Event status

Example Request

# Update only status
curl -X PUT https://m4vehfzim8.execute-api.us-west-2.amazonaws.com/prod/events/614b345c-5114-4788-bd81-f792d59463b5 \
  -H "Content-Type: application/json" \
  -d '{"status": "cancelled"}'

# Update multiple fields
curl -X PUT https://m4vehfzim8.execute-api.us-west-2.amazonaws.com/prod/events/614b345c-5114-4788-bd81-f792d59463b5 \
  -H "Content-Type: application/json" \
  -d '{
    "status": "active",
    "capacity": 600
  }'

Response

200 OK - Returns the complete updated event

{
  "eventId": "614b345c-5114-4788-bd81-f792d59463b5",
  "title": "Tech Conference 2024",
  "description": "Annual technology conference",
  "date": "2024-12-15",
  "location": "San Francisco",
  "capacity": 600,
  "organizer": "Tech Corp",
  "status": "active"
}

Error Responses

404 Not Found - Event does not exist

{
  "detail": "Event with id '614b345c-5114-4788-bd81-f792d59463b5' not found"
}

422 Unprocessable Entity - Validation error

{
  "detail": "Validation error message"
}
